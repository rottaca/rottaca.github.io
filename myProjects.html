3DScanner

3DScanner is an implementation of a 3D object scanner system. It is based on cheap usb webcams, a small rotatable platform and a microcotroller controller that is controlled by the computer. In the future it will be able to transform small realworld objects into 3d pointclouds. The software is based on my computer vision library FreeCV and uses Qt.
